基于eclipse插件的JeeSite框架代码生成器

程序安装方法：
将dist/plugins/com.thinkgem.jeesite.autocode_1.0.0.beta.jar插件文件拷贝至eclipse安装目录下的plugin目录中
重启eclispe，在菜单栏将看到JEESITE菜单.

提供增删改查、报表、树状结构、主从表、excel导入文件功能代码的生成。
